# Начать учиться    
Это учебная проектная работа, представляющая собой одностраничный сайт. Здесь собраны современные и эффективные подходы к обучению. Реализован с помощью HTML и CSS.
## Технологии
Здесь используются:
* флексбокс-вёрстка;
* анимация;
* плавность переходов;
* кейфрейм;
* YouTube API;
* организация файловой структуры Nested
